2025년 1월 약 $2B에서 시작한 토큰화 미국채 시장이 2026년 4월 $12.88B 사상 최대. 같은 기간 토큰화 RWA (Real World Assets; 실물자산 토큰화 전체) 시장은 $19.3B로 +256% 폭증. 어제 JP모건·리플·마스터카드의 첫 cross-border 정산 성공이 다음 단계의 시작점
Source: rwa.xyz · TheStreet Crypto · Q1 2026 기준
TOKENIZATION · CROSS-BORDER
🔗 토큰화 미국채, 첫 cross-border 정산이 5초 만에 끝났음
◾Ondo Finance (Goldman·Coinbase 출신이 차린 토큰화 RWA 플랫폼)가 JP모건의 Kinexys (JP모건의 블록체인 결제 인프라) + 마스터카드 MTN (Multi-Token Network; 온체인 자산과 fiat의 상호운용 layer) + Ripple과 함께 토큰화 미국채의 cross-border, cross-bank 정산 첫 성공 (5/6 발표). XRP Ledger에서 5초 미만 처리
◾거래 흐름이 핵심. Ripple이 본인 보유한 OUSG (Ondo Short-Term U.S. Government Treasuries; 단기 미국채 토큰)를 환매 → 마스터카드 MTN이 fiat 정산 명령을 라우팅 → JP모건 Kinexys가 Ondo 블록체인 예금 계좌 차감 → 싱가포르 Ripple 은행 계좌로 USD 입금. 종전 t+1·t+2 정산이 한 흐름 5초로 압축
◾의의는 정통 결제망과 공공 블록체인이 단일 흐름으로 맞물린 첫 케이스. 종전 토큰화 자산 환매는 "블록체인은 즉시지만 fiat 정산은 wire + 수기 + 영업시간"으로 분리되어 있었는데, 이번엔 블록체인 거래가 fiat 정산을 직접 트리거. 24/7 글로벌 시장 인프라의 시제품
◾XRP Ledger의 RWA 잠겨있는 가치 $25억+ 돌파. 토큰화 미국채 시장 자체는 4월 초 $128.8억으로 15개월 +225%. JP모건은 자체 토큰 JPM을 Base 네트워크에 발행했고, 다음엔 Canton 네트워크로 확장 예정. 정통 결제망 4개사가 동시에 토큰화 인프라 진입
◾Ondo의 Ian De Bode 사장 — "공공 블록체인과 글로벌 은행 인프라가 cross-border를 실시간으로 같이 정산한 첫 사례". 다음 단계는 일반 증권계좌 사용자가 토큰화 자산 직접 거래 가능 환경. 어제 Bullish의 Equiniti 인수 + FINRA의 Securitize 그린라이트와 결합되면 "토큰화 증권의 발행·정산·등록 풀스택"이 한 번에 완성되는 그림
🏛 전통 금융 대비 현재 위치
정통 SWIFT cross-border wire 1~3일 + 수십불 수수료 + 영업시간만 가능 vs 이번 파일럿 5초 + 거의 0 수수료 + 24/7 — 같은 결제 행위인데 시간·비용이 차원이 다른 구조
AI · STABLECOIN
🤖 솔라나·구글 클라우드, AI 에이전트용 결제 게이트웨이 첫 출시
◾Solana Foundation + Google Cloud, AI 에이전트가 stablecoin으로 API 호출당 결제하는 게이트웨이 Pay.sh 정식 출시 (5/5 발표). Gemini, BigQuery, Vertex AI, Cloud Run + 50+ 외부 API 지원. 호출당 약 $0.005, 월정액 구독 대신 "쓴 만큼만" 결제. 에이전트는 Solana 지갑 하나만 만들면 끝
◾핵심 인프라는 x402 프로토콜 — 코인베이스가 인큐베이팅했고 지금은 Linux Foundation이 표준 관리. HTTP 402 "Payment Required" 상태 코드 기반으로, AI 에이전트가 계좌·KYC·구독 없이 자기 지갑으로 직접 결제. AWS·Visa·Mastercard도 x402 표준에 합류. 머신 간 결제 (M2M)의 첫 본격 인프라
◾의의는 구글이 본인 클라우드의 핵심 API 결제 인프라로 솔라나를 지정한 것. 구글은 5월 들어 Solana validator 노드도 운영 시작. 정통 빅테크가 특정 퍼블릭 블록체인을 결제 백본으로 채택하는 첫 케이스 — "구글이 Solana 위에 결제 인프라를 깐다"는 메시지가 시장에 줄 신호가 큼
◾Stripe의 Tempo가 만든 Machine Payments Protocol도 Pay.sh가 동시 지원. 두 표준 모두 호환되는 결제 인프라. 이미 x402 단독으로만 1.65억건 거래 + 48만 에이전트가 활동 중인 상태. 시장이 빠르게 형성되는 중 — 6월부터는 일반 개발자도 본인 API를 등록하고 호출당 받기 가능
◾이전엔 트레이더·DeFi가 stablecoin 주 사용처였는데, 머신 간 결제로 사용처 확장. 솔라나는 발표 직후 SOL 토큰이 $90 도전 모드. 어제 다룬 Western Union USDPT (송금) + Mastercard Rain principal 등록 (카드)에 더해 클라우드 결제까지 — stablecoin 사용처가 한 주 만에 4영역으로 확장
🏛 전통 금융 대비 현재 위치
정통 SaaS는 월정액 + 카드 결제 + 사람 KYC가 기본 vs Pay.sh는 AI 에이전트가 자기 지갑으로 호출당 1초 안에 결제 — 결제 단위가 "월/사람"에서 "초/머신"으로 바뀌는 중
BROKERAGE · CRYPTO TRADING
💼 모건스탠리, E*Trade 크립토 거래 50bp로 가격 정조준
◾모건스탠리가 E*Trade (2020년 130억불에 인수한 자산관리 플랫폼)에서 BTC·ETH·SOL 거래를 50bps (0.5%) 수수료로 출시 (5/6 발표). 코인베이스 60bp + 로빈후드 95bp 가격 직접 정조준. 850만 자산보유 가입자가 즉시 거래 가능
◾모건스탠리 자산관리 헤드 Jed Finn — "이번 출시는 단순히 더 싼 가격으로 크립토를 거래하는 것 그 이상". 정통 브로커리지가 크립토를 자기 플랫폼에 직접 깔면, 자산 관리 자체가 가상자산을 포함하는 새 단계. 본인이 사용한 표현 — "탈중개자들의 탈중개 (disintermediating the disintermediators)"
◾의의는 코인베이스의 비즈니스 모델 핵심 (거래 수수료)이 정조준 받기 시작. 코인베이스는 같은 주 14% 감원 발표 + 매니저층 평탄화 + 'player-coach' 모델 도입 (직접 일하는 매니저)로 전환. 가격 경쟁자 + 비즈니스 모델 압박이 동시에 들어옴 — 어제부터 시장이 이걸 "AI Washing 아니라 진짜 본업 위협"으로 다시 보는 중
◾E*Trade 850만 사용자 모두 이미 KYC + 미국 증권계좌 보유 상태. 즉시 크립토 거래로 전환 가능 — 코인베이스의 핵심 우위 ("유저 락인" + 시장 점유율)가 직접 흔들림. 코인베이스 주가는 발표 직후 +5% 반응 (시장은 14% 감원을 비용 절감으로 해석)이지만, 중장기 매출 압박은 별도 이슈
◾같은 시점에 UBS도 XRP 보유 공시 후 크립토 거래 출시 예정 + 트럼프 일가 World Liberty도 자체 거래소 검토 중. "글로벌 1위 자산관리사들이 크립토를 본인 플랫폼에 옮기기 시작"이 5/6 한 주의 핵심 결. 코인베이스 입장에서 다음 분기 거래량·수수료율 동시 압박이 불가피한 상황
🏛 전통 금융 대비 현재 위치
코인베이스 거래 수수료 60bp + 자체 마진 vs 모건스탠리 50bp + 850만 KYC 완료 유저 — 같은 거래 서비스를 정통 금융이 가격·유저 풀 양쪽으로 우위에 서서 진입
TOKENIZATION · M&A
🏢 Bullish가 영국 Equiniti $42억에 인수 — "토큰화 transfer agent"
◾미국 가상자산 거래소 Bullish (2021년 창업, Tom Farley CEO; 전 NYSE 대표)가 영국 Equiniti를 42억불에 인수 (5/5 발표). Equiniti는 1929년 창업한 영국 1위 transfer agent (상장사 주주명부 관리·배당 처리·증권 등록 기관). FTSE 100 절반 이상이 Equiniti 고객

◾인수 자금은 자체 IPO + 토큰화 채권 결합. Bullish는 작년 NYSE 본격 상장 — 가상자산 거래소가 정통 금융 인프라 인수에 자기 주식을 통화로 사용한 첫 케이스. 5년 회사가 백년 회사를 사들이는 시대 전환의 명백한 시그널

◾다음 단계는 토큰화 IPO의 본격 시작. FINRA가 Securitize에 토큰화 IPO 인수·보관 자격 그린라이트 (지난주). 어제 JP모건·Ripple·Mastercard의 토큰화 미국채 정산 + Bullish의 Equiniti 인수가 합쳐지면, 토큰화 증권의 발행·정산·등록 인프라가 한 번에 완성되는 그림 — 다음 분기 토큰화 IPO 1호가 시간 문제

Tokenization (RWA · Real-World Asset Tokenization)
"실물자산 토큰화." 미국채·주식·부동산·채권·재보험 등 정통 금융 자산을 블록체인 위 토큰으로 발행해 24/7 거래·정산하는 시스템. 어제 JP모건-Ripple-Mastercard의 토큰화 미국채 정산이 첫 cross-border 케이스. 토큰화 미국채 시장 4월 $128.8억으로 15개월 +225%, 전체 RWA $193억
2026 가상자산의 핵심 키워드

"For the first time, a public blockchain and global banking infrastructure settled a cross-border transaction of a tokenized fund together in real time."
Machine-to-Machine (M2M) Payment / x402
"머신 간 결제." AI 에이전트가 사람 개입 없이 자기 지갑으로 API 호출당 결제하는 시스템. x402는 HTTP 402 "Payment Required" 상태 코드를 표준화한 프로토콜 — 코인베이스 인큐베이션, 지금 Linux Foundation이 표준 관리. 솔라나 Pay.sh가 첫 본격 인프라. AWS·Visa·Mastercard·Stripe·Google이 같은 표준 채택

Disintermediation
"중개 단계 제거." 2008년 비트코인 탄생 시점부터 가상자산이 자처한 명분 — 은행·증권사·결제망 같은 중간 단계를 없애고 사용자끼리 직접 거래. 모건스탠리 자산관리 헤드 Jed Finn이 어제 E*Trade 크립토 거래 출시 발표하며 사용한 표현 "탈중개자들의 탈중개" — 가상자산이 정통 금융을 disrupt하던 흐름이 이제 정통 금융이 가상자산 거래소를 disrupt하는 단계로
2008년 자처한 명분이 2026년 부메랑으로
